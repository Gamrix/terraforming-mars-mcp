# Next Game Protocol

Use this before every new game and during key checkpoints.

The purpose is to force actual improvement game over game.
Do not start from intuition alone.

## 1. Pre-Game Commitments

Before the first meaningful action, commit to these rules:

- I will not use `hybrid` as a vague default.
- I will forecast final scoring categories by midgame.
- I will react to opponent engine alarms immediately, not one generation later.
- I will run a scripted Gen 10+ liquidation plan.
- If Mars becomes fully terraformed mid-generation, I will verify whether actions continue before passing.

## 2. Gen 3 Checkpoint

Write down:

- Macro plan:
- `hard close`
- `hybrid leaning close`
- `outscale`

- Why this plan is valid:
- Which categories should I expect to win?
- Which milestone races are real?
- Which award lanes are likely live?

If the macro call cannot be justified in one short paragraph, it is probably too vague.

## 3. Gen 6 Forecast

Estimate both players' likely final scores in:

- `TR`
- `awards`
- `greenery`
- `city`
- `card_vp`

Then answer:

- If the game lasts one extra generation, who benefits more?
- Am I ahead only because of TR?
- Does my current line actually beat John's projected `card_vp + city` total?

If the answer is "no" or "unclear", revise the plan immediately.

## 4. Opponent Alarm Thresholds

These are hard alarms, not vague warnings.

- If John has `AI Central + Mass Converter`, assume his engine is live.
- If John adds `Quantum Extractor`, `Anti-Gravity Technology`, `Shuttles`, `Earth Office`, or another discount on top of that, treat it as a closure emergency.
- If projected opponent `card_vp >= 35` by Gen 8 or `>= 40` by Gen 9, stop generic setup and move to closure or direct VP denial.
- If a funded award flips against me by Gen 9 to 10, stop defending it emotionally and reallocate actions.

## 5. Gen 8 Recommit

At the start of Gen 8, rewrite:

- Current macro plan.
- Current final-score forecast.
- Award contest-or-concede decision.
- Exact estimate of remaining generations.
- Which two actions I most likely want this generation.

If this is not written, the game is drifting.

## 6. Gen 10+ Liquidation Script

At the start of every late generation:

- Best `MC -> VP` conversion:
- Best `plants -> greenery` conversion:
- Best `city + adjacency` conversion:
- Best direct closure line:
- Best denial line:
- Which resources will strand if I pass now?

Then answer:

- Can Mars end this generation?
- Should Mars end this generation?
- If Mars ends now, does the action phase still continue?

Do not pass until these are answered.

## 7. Post-Terraform Rule

If Mars becomes fully terraformed during the action phase:

- Poll the prompt.
- Assume more actions may still be legal.
- Spend remaining `MC`, `steel`, `titanium`, `plants`, and live card actions on VP before passing.
- Only then rely on final greenery cleanup.

## 8. Active Hypotheses

Keep only 1 to 3 live hypotheses across games.

Good examples:

- Earlier city anchors improve city delta.
- Fewer funded awards improve net VP.
- Hard-close commitment by Gen 8 performs better than hybrid drift against John's science engine.

After each game, mark each hypothesis:

- `strengthened`
- `weakened`
- `unresolved`

If a rule is not producing better outcomes after several games, rewrite or replace it.
